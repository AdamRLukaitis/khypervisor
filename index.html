Welcome to Khypervisor git.

Khypervisor is an open-source type-1 hypervisor, which aims at providing a monolithic, light-weight, portable, and flexible virtualization solution. It provides a high performance for ARMv7 Virtualization Extensions.

Khypervisor primarily supports Full virtualization hence, supports a wide range of unmodified Guest operating systems such as Linux and RTOS, and the Bare Metal Application.

It has most features expected from a modern hypervisor, such as: IO device emulation framework, Pass through hardware access and many more.

Khypervisor is Hardware Accelerated Hypervisor, so it supprots ARM GICv2, ARM Generic Timer and 2-Stage Address Translation.

Features

Features Hardware Acclerated Hypervisor

  1. ARMv7 Virtualization Extensions
  2. Hyp Mode, 2-Stage Address Translation, and GICv2
Full Virtualization
  1. No Guest Modification Required
  1. Device Dedication Model – Each VM Reserves Exclusive Access to Dedicated Hardware Devices
Supported Guest OS Types: Linux, RTOS and Bare Metal Application Supported architecture : ARM Cortex-A15 Supported Target Hardware
  1. ARM RTSM VE - FastModels
  2. InSignal’s Arndale Board – Exynos 5250
